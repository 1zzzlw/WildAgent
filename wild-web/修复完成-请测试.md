# ✅ 渲染引擎Bug修复完成

## 已修复的问题

### 1. 🔴 墙体几何异常（关键bug）
- **问题**：墙体边界盒巨大（-13.75到8.5），模型错乱交叉
- **原因**：开口坐标系统错误，世界坐标被当作沿墙距离
- **修复**：正确计算开口沿墙投影距离
- **文件**：`src/wild-core/src/primitive/resolver.ts`

### 2. 🔴 屋顶位置丢失
- **问题**：屋顶不在墙顶，position变成[0,0,0]
- **原因**：瓦片合并逻辑错误地合并了屋顶主体
- **修复**：添加elementId检查，避免屋顶被合并
- **文件**：`src/wild-core/src/primitive/index.ts`

### 3. 🟡 网格线消失
- **问题**：GridHelper显示一瞬间就消失
- **原因**：场景更新时没有保持GridHelper引用
- **修复**：保持引用并在更新时检查
- **文件**：`src/components/viewport/CanvasViewport.vue`
