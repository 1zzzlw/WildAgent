## 常见问题

### Q: 还是看到窗户飘在墙外面？
A: 请确认：
1. opening.ts 的修改已保存并重新构建
2. 刷新浏览器缓存 (Ctrl+F5)
3. 检查控制台是否有JavaScript错误

### Q: 还是看到地板下有空隙？
A: 请确认：
1. 已重新构建项目 (`npm run build` 或重启 `npm run dev`)
2. 刷新浏览器缓存 (Ctrl+F5)
3. 检查是否加载了正确的蓝图文件

### Q: 二层窗户还是有问题？
A: 检查：
1. resolver.ts 的修改已保存
2. 蓝图文件中 `from[1]` 使用的是世界Y坐标
3. 浏览器控制台的cutout日志，`localY` 应该是相对墙底的偏移

### Q: 想验证修复是否生效？
A: 在浏览器控制台查看日志：
```
🧱 buildWall wall_upper_front
  cutouts: 1
    - localX=8.00, localW=2.00

📦 boxWithHoles: ...
  转换后的holes:
    [0] ... y1=-0.40, y2=1.10  // 这是相对墙中心的坐标
```

如果看到 `localY` 是小数值（如1.0），说明修复成功。  
如果还是大数值（如4.0），说明修复未生效。
